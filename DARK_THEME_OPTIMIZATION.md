## 技术实现

### CSS选择器策略
使用 `[data-theme="dark"]` 属性选择器来应用深色主题样式：

```css
[data-theme="dark"] .component {
  background: #141414;
  color: #fff;
  border-color: #303030;
}
```

### 颜色规范
- **主背景色**：`#000` (页面背景)、`#141414` (组件背景)
- **次级背景色**：`#1f1f1f` (卡片、面板)、`#262626` (悬停状态)
- **边框颜色**：`#303030` (主边框)、`#434343` (次级边框)
- **文字颜色**：`#fff` (主文字)、`#8c8c8c` (次级文字)
- **强调色**：`#177ddc` (链接、激活状态)

### 组件适配
- **Ant Design组件**：通过 `:deep()` 选择器覆盖组件内部样式
- **自定义组件**：直接应用深色主题类名
- **滚动条**：统一的深色滚动条样式
